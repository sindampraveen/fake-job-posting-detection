# Fake Job Posting Detection

Overview

This project detects fake job postings using Machine Learning techniques by analyzing linguistic and metadata features.

Features

- Detects fraudulent job postings
- Data preprocessing and feature extraction
- Machine Learning based classification
- User-friendly interface

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Tkinter

Project Type

Team Project developed as part of our college curriculum.

Team Contribution

This project was completed collaboratively by our team, and I actively participated in the project development and implementation.
